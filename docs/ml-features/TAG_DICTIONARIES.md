## Adding New Tags

### Contribution Guidelines

1. **Identify gap**: Find common words in user notes not being detected
2. **Categorize**: Determine which category (taste, dish, cuisine, etc.)
3. **Add keywords**: Include variations, synonyms, and common misspellings
4. **Test**: Verify extraction works with real notes
5. **Submit PR**: Update this file + `tagExtractor.ts`

### Example Addition

```typescript
// In lib/ml/tagExtractor.ts
const TASTE_DESCRIPTORS = {
  // ... existing tags
  
  // NEW TAG
  tangy: ['tangy', 'zesty', 'zingy', 'bright', 'punchy']
};
```

Then update this document with the new tag entry.

## Tag Confidence Scoring

**How confidence is calculated:**

```typescript
confidence = min(keyword_matches * weight, 1.0)
```

**Weight values:**
- Taste profiles: 0.3 per match
- Dish types: 0.4 per match
- Quality indicators: 0.5 per match

**Example:**
```
Notes: "spicy hot chile jalapeño sauce"
Matches: 4 keywords for "spicy"
Confidence: min(4 * 0.3, 1.0) = 1.0 (100%)
```
